物性物理 有名物質俯瞰マップ

量子相関の極限、トポロジーの数理、そして非自明な超伝導秩序。
日本の研究者がリードする現代物理学の最前線を網羅する。

1. 量子臨界性と非従来型超伝導

強磁性超伝導 (FM-SC)

URhGe / UCoGe / UGe21:1:1 / 1:2

強磁性秩序と超伝導が同一の電子（f電子）によって担われ、ミクロに共存する稀有な系。通常、磁場は超伝導を破壊するが、これらウラン系では磁気秩序そのものが超伝導対形成の「糊」となる。

物理学的解釈： 強磁性量子臨界点近傍での「縦スピンゆらぎ」が引力となり、スピン三重項対（S=1）が形成される。URhGeではb軸方向に強い磁場を印加することで強磁性ゆらぎが極大化し、超伝導が復活する「再入超伝導」現象が実証されている。
青木 大 (東北大) / 石田 憲二 (京大) / 佐藤 憲昭 (名大名誉)
UTe21:2

2019年に発見された重い電子系超伝導。非磁性だが強磁性量子臨界点に極めて近く、カイラルp波（三重項）のトポロジカル超伝導候補として世界的に注目。

物理学的解釈： 近藤効果によるc-f混成の結果、自由電子の数百倍の有効質量を持つ準粒子が形成される。60テスラを超える強磁場下でも超伝導相が安定であり、マヨラナフェルミオンの存在を示唆する実験結果も報告されている。
青木 大 (東北大) / 石田 憲二 (京大) / 徳永 陽 (JAEA)

鉄系超伝導 (Iron-based SC)

LaFeAsO1-xFx / BaFe2As21111 / 122

2008年に日本で発見された高温超伝導体。FeAs層における多軌道性が物理の核心であり、スピンゆらぎと軌道ゆらぎの競合によって対形成が行われる。

物理学的解釈： 対称性はs±波が有力。異なるフェルミ面間で超伝導ギャップの符号が反転する非自明なs波状態である。中性子散乱による共鳴ピークの観測が、この符号反転の直接的な証拠として提示されている。
細野 秀雄 (東工大) / 廣井 善二 (東大) / 芝内 孝禎 (東大)
FeSe1:1

鉄系の中で最も単純な構造だが、物理は最も複雑。長距離磁気秩序を欠く一方で、電子系が回転対称性を破る「ネマティック相」を示す。

物理学的解釈： 軌道自由度が主役となる「軌道ネマティック秩序」が超伝導と密接に絡み合う。単層膜化（FeSe/SrTiO3）により転移温度が劇的に上昇（最高100K超の報告あり）するなど、界面物理の舞台でもある。
松田 祐司 (京大) / 芝内 孝禎 (東大)

2. 実空間・運動量空間の幾何学

カゴメ格子とベリー曲率

Mn3Snカゴメ格子

磁気モーメントが正三角形の頂点で120度ずつ向き合う「非共線的反強磁性」を室温で維持する。マクロな磁化をほとんど持たないため、漏れ磁場による干渉が少ない。

物理学的解釈： 反強磁性構造によって生じる実空間のワイル点（Weyl Nodes）が、運動量空間に強力なベリー曲率を発生させる。これが「仮想磁場」として作用し、室温での巨大異常ホール効果や異常ネルンスト効果を誘発する。
中辻 知 (東大) / 大谷 義近 (理研/東大)
MnSiスカイミオン

結晶の反転対称性の欠如（B20構造）により、スピンの渦巻き構造である「磁気スカイミオン」が安定化する代表的物質。

物理学的解釈： Dzyaloshinskii-Moriya（DM）相互作用が起源。スカイミオンはトポロジカルに保護された量子化磁束のような性質を持ち、極めて微弱な電流（従来の10万分の1）での駆動が可能なため、究極の省エネ情報デバイスとして期待されている。
十倉 好紀 (東大/理研) / 加倉井 和久 (JAEA)

トポロジカル相と界面物性

Bi2Se3 / (Bi,Sb)2Te32:3系

強いスピン軌道相互作用によりバンド構造が反転した「トポロジカル絶縁体」。表面にはスピンと運動量が直交する「ヘリカル・ディラック・コーン」が現れる。

物理学的解釈： 表面の伝導電子は時間反転対称性によって保護されており、非磁性不純物による後方散乱が禁じられている。これを磁性体と接合させることで、量子異常ホール効果などの非自明な現象が発現する。
安藤 陽一 (大阪大/ケルン大) / 金道 浩一 (東大)
Sr2RuO4 (SRO)2:1:4系

ルテニウム酸化物の層状ペロブスカイト。純度が極めて高い単結晶において超伝導が発現し、スピン三重項の有力候補として20年以上君臨している。

最新論争： 最近のNMRナイトシフト実験において、超伝導転移に伴うスピン磁化率の減少が再確認されたため、従来のカイラルp波説が揺らいでいる。現在、スピン一重項d波説や他の三重項状態の可能性を含めた国際的な再検証の渦中にある。
前野 悦輝 (京大) / 石田 憲二 (京大)

3. 対称性の自発的破れと多体効果

モット絶縁体と量子ゆらぎ

V2O3 / κ-(BEDT-TTF)2X強相関

電子同士のクーロン排斥力（U）がバンド幅（W）を上回り、本来金属であるはずが絶縁体化する現象。強相関物性の原点。

物理学的解釈： 温度、圧力、キャリアドープによりU/Wを制御することで、金属・絶縁体相転移（モット転移）が引き起こされる。有機導体（κ体）では、転移点近傍でスピン液体状態や超伝導、強誘電性が複雑に絡み合う。
今田 正俊 (東大名誉) / 鹿野田 一司 (東大) / 加藤 礼三 (理研)

誘電・焦電・圧電材料

Pb(Zr,Ti)O3 (PZT) / BaTiO3ペロブスカイト

構造的な反転対称性の欠如により自発分極が生じる。PZTは、圧電定数が極めて大きく、世界中のセンサ・アクチュエータの基盤材料。

物理学的解釈： ソフトモードの凍結による相転移。特に菱面体晶と正方晶の相境界（MPB）では、分極ベクトルが容易に回転できるため、外部応力や電場に対して巨大な応答を示す。物性的にはナノスケールの分域（ドメイン）構造の制御が鍵。
内野 研二 (ペンシルバニア大/上智大)

専門用語・注釈 (Glossary)

量子臨界点 (QCP) 絶対零度において磁気相転移などの相転移温度が0Kに達する点。量子力学的なゆらぎが極大化し、非従来型超伝導の起源となる。
スピン三重項 (Spin Triplet) 電子ペアのスピンが揃った状態（S=1）。反対称な軌道関数を要求するため、p波やf波といった非自明な超伝導ギャップを持つ。
ベリー曲率 (Berry Curvature) 波動関数の幾何学的位相から導かれる量。運動量空間における「磁場」として働き、外部磁場なしの異常ホール効果を支配する。
ネマティック秩序 (Nematic Order) 並進対称性を保ちながら、x軸方向とy軸方向が等価でなくなるような回転対称性の自発的破れ。電子液晶状態とも呼ばれる。
重い電子系 (Heavy Fermion) 局在性の強いf電子と伝導電子が混成し、低温で電子の有効質量が自由電子の100～1000倍程度まで増大する多体系。
ナイトシフト (Knight Shift) NMR測定における共鳴周波数のズレ。電子系のスピン磁化率を直接反映するため、超伝導対の状態判定に必須のデータ。
Dzyaloshinskii-Moriya (DM) 相互作用 反転対称性がない系でスピンをひねらせようとする非対称交換相互作用。スカイミオン形成の原動力。
多極子秩序 (Multipole Order) スピンの向きだけでなく、電子の軌道の形や電荷分布の歪みが整列する秩序。通常の磁気測定では見えにくいため「隠れた秩序」と呼ばれる。
